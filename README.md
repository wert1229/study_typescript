# 유데미 강의 실습
## Understanding Typescript - 2021 Edition

### Core Types
* number -> 1, 5.3, -10
* string -> 'hi', "hi", \`hi\`
* boolean -> true, false
* object -> {age : 30}
* Array -> [1, 2, 3]
* Tuple -> [1, '2']
* Enum -> enum {New, OLD}
* Any -> *

* Union -> string | number
* Literal -> 'One' | 'Two'
* Function -> (num: nubmer) => void
* unknown
* never
